# 백신 챙겨 (Vaccine Chaeng-gyeo)

성인 예방접종 결정 도우미 — 흩어진 지침을 **당신의 상황 하나로** 정리해 '지금 챙길 백신'만 보여주는 정적 웹앱.

> 철학: 정보를 **더하지** 않고 **덜어내서**, 입력 → 한 화면 결정.

## 실행
`index.html` 파일 하나. 더블클릭으로 브라우저에서 바로 열립니다. 백엔드·빌드 없음.

## 배포 (URL 확보 = 제출 요건)
셋 중 아무거나:
- **Vercel** — [vercel.com/new](https://vercel.com/new) 에 이 폴더 드래그 → 즉시 URL. (또는 `npx vercel`)
- **Netlify** — [app.netlify.com/drop](https://app.netlify.com/drop) 에 폴더 드래그.
- **GitHub Pages** — repo 푸시 후 Settings → Pages → main 브랜치.

## 구조
- **규칙 엔진** (`VACCINES` 배열): 나이·기저질환·접종이력 → `지금/나중/완료` 분류. 의료 안전을 위해 LLM이 아닌 **규칙 기반**.
- **자연어 자동 채우기**: "63세 당뇨, 대상포진 안 맞음" 같은 문장을 클라이언트 휴리스틱으로 파싱해 입력 자동화. (2단계에서 LLM으로 대체 가능)
- **화면**: 홈 → 입력(1화면) → 결과(1화면).

## 백신 규칙 (⚠️ 임상 감수 필요)
질병청·대한감염학회 성인 예방접종 권고 요약. 인플루엔자 / 폐렴구균(PPSV23) / 대상포진 / Tdap·Td / A형간염 5종.
연령컷·조건·간격의 최종 기준은 [예방접종도우미](https://nip.kdca.go.kr)에서 검증하세요.
